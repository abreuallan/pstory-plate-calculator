# pstory plate calculator

A lightweight web calculator for Pokémon training plates with deterministic and Monte Carlo scenarios.

## Running locally

Open `web/index.html` in your browser, or serve the folder with a simple static server.

## Features

- Simulates 100% success, 100% fail, and Monte Carlo average scenarios.
- Uses a lightweight HTML, CSS, and JavaScript interface.
- Deployed through GitHub Pages.

## Project structure

- `web/index.html` — user interface.
- `web/styles.css` — styles.
- `web/script.js` — calculation logic.

## Note

This project is now a web-only app with no backend or extra dependencies.
